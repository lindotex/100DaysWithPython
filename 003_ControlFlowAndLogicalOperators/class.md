# Conditional Statements, Logical Operators, Code Blocks and Scope.

If/Else statement:

```python
    if condition:
        do this
    else:
        do that
```

example:

```python
# Controling the level of water.
water_level = 50
if water_level > 80;
    print("Drain Water")
else:
    print("Continue")
```
